# LLM Context

Personal knowledge base and Obsidian vault, synced automatically every 4 hours. Use this repo to give any LLM full context about ongoing work, preferences, and ideas.

## Structure

| Folder | Contents |
|---|---|
| `user/` | Profile, workflow, writing style |
| `feedback/` | Corrections and behavior rules for LLMs |
| `projects/` | Active and past projects with details, goals, and status |
| `ideas/` | Ideas and concepts shared across sessions |
| `rules/` | Instructions and rules for LLMs to follow |
| `references/` | Articles, interview prep, external resources |

## How to use with an LLM

Paste the relevant files or link to this repo at the start of a conversation:

```
Here is my context repo: https://github.com/lrshaid/llm-context
Please read the files in user/, rules/, and any relevant project files before we begin.
```

## Obsidian

This repo doubles as an Obsidian vault. Open the root folder in Obsidian and use `_MOC.md` as the home page.
