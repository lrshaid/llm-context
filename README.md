# LLM Context

Personal knowledge base synced automatically every 4 hours. Use this repo to give any LLM full context about ongoing work, preferences, and ideas.

## Structure

| Folder | Contents |
|---|---|
| `memory/` | Claude Code persistent memories (user profile, feedback, project state) |
| `projects/` | Active and past projects with details, goals, and status |
| `ideas/` | Ideas and concepts shared across sessions |
| `rules/` | Instructions and rules for LLMs to follow |
| `code-examples/` | Useful snippets, patterns, and reference code |

## How to use with an LLM

Paste the relevant files or link to this repo at the start of a conversation:

```
Here is my context repo: https://github.com/lrshaid/llm-context
Please read the files in memory/, rules/, and any relevant project files before we begin.
```
